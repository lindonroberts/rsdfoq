======================================================
RSDFO-Q: Randomized Subpsace DFO with Quadratic Models
======================================================

A Python package for general minimization, where derivatives are not available, using random subspaces.
For a description of this algorithm, see [this paper](https://arxiv.org/abs/2412.14431).

For lower-dimensional problems, consider using the more actively maintained [Py-BOBYQA](https://github.com/numericalalgorithmsgroup/pybobyqa).
